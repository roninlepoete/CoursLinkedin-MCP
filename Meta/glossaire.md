## 🔧 TERMINOLOGIE TECHNIQUE SPÉCIFIQUE WS

### Chaînage d'informations
**Définition** : Flux d'information entre modules/systèmes dans WS.

**Exemple AudioTools** : Audio → FFmpeg → Transcription → Analyse → Markdown

---

### fal.ai
**Définition** : Plateforme IA GPU Cloud utilisée dans Mikhaelize pour génération images/vidéos.

**API** : Serverless, modèles pré-entraînés (fast-sdxl, etc.)

---

### Idempotence (concept trigger-word)
**Définition** : Exécution N fois = même résultat qu'1 fois.

**Importance** : Garantit sécurité re-exécution scripts/commandes.

---

### MCP (Model Context Protocol)
**Définition** : Protocole communication IA utilisé par Claude Desktop et GitHub Copilot.

**WS** : Plusieurs MCP servers (filesystem, context-injection, desktop-commander, github, memory, docker)

---

### STB (Storyboard Mikhaelize)
**Définition** : Transformation paroles musicales → timeline visuelle avec descriptions images par timestamp.

**Format** :
```
[00:00] - Description scène 1
[00:15] - Description scène 2
```
